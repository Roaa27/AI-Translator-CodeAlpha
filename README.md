# 🌍 AI Language Translator

A simple and powerful **AI-based language translator** built using **Streamlit**, supporting multiple languages with text-to-speech and translation history.

---

## 🚀 Live Demo

👉 Try the app :
https://ai-translator-codealpha-ahsthalhxjwbxck9lzctjo.streamlit.app/

---

## ✨ Features

* 🌐 Translate text instantly between multiple languages
* 🗣️ Text-to-Speech (listen to translated text)
* 📜 Translation history (last 5 translations)
* 🧹 Clear history with one click
* 🎨 Clean and simple UI using Streamlit
* ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

* **Python**
* **Streamlit**
* **deep-translator**
* **gTTS (Google Text-to-Speech)**

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/ai-translator.git
cd ai-translator
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```
